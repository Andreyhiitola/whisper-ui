# DeepSeek API

DeepSeek — китайская AI-компания, модели уровня GPT-4o по качеству анализа.

## Роль в проекте

Опциональный провайдер для анализа разговора. Выбирается в интерфейсе вместо Groq когда нужно более качественное и развёрнутое AI-заключение.

OCR картинок при выборе DeepSeek всё равно идёт через Groq Vision — DeepSeek не имеет Vision API в данной интеграции.

## Сравнение с Groq

| | Groq | DeepSeek |
|---|---|---|
| Цена анализа | Бесплатно | ~$0.0003/запрос |
| Скорость | Очень быстро (LPU) | Медленнее (GPU) |
| Качество анализа | Хорошее (Llama 3.3 70b) | Выше (уровень GPT-4o) |
| OCR картинок | Groq Vision | Groq Vision |

## Когда использовать DeepSeek

- Сложные сценарии с большим количеством пунктов
- Нужен детальный структурированный отчёт
- Анализ тональности и поведенческих паттернов
- Groq даёт недостаточно точный результат

## Получить ключ

1. [platform.deepseek.com](https://platform.deepseek.com) → зарегистрироваться
2. Пополнить баланс от $2
3. Меню → **API Keys** → **Create API Key**
4. Ключ начинается с `sk-...`
5. Вставить в настройки интерфейса — сохраняется в localStorage браузера

## Стоимость

- `deepseek-chat`: $0.14 / млн входящих токенов, $0.28 / млн исходящих
- Один анализ разговора (~2 000 токенов) ≈ **$0.0003**
- $2 хватит примерно на **6 000 анализов**

## Модели

| Модель | Описание |
|--------|----------|
| `deepseek-chat` | Универсальный, оптимальный выбор |
| `deepseek-reasoner` | С цепочкой рассуждений, медленнее но точнее |
