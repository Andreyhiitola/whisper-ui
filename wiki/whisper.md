# Whisper ASR

OpenAI Whisper — модель распознавания речи. Запускается локально через Docker.

## Модели

| Модель | Размер | Скорость на CPU | Качество русского |
|--------|--------|-----------------|-------------------|
| tiny   | 75 MB  | очень быстро    | плохое |
| base   | 145 MB | быстро          | приемлемое |
| **small** | **460 MB** | **средне** | **хорошее** ← текущая |
| medium | 1.5 GB | медленно        | отличное |

## Изменить модель

В `docker-compose.yml`:
```yaml
environment:
  - ASR_MODEL=small  # заменить на нужную
```

После изменения: перезапустить контейнер в OMV или `docker compose up -d`.

Модель скачивается при первом запуске и кэшируется на диске.

## API эндпоинты

```
POST /asr?language=ru&output=txt   — транскрибация
POST /detect-language              — определить язык
```

Параметр `output`: `txt`, `srt`, `vtt`
