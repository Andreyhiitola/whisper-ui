# Groq API

Groq — облачный сервис для быстрого инференса LLM на кастомных чипах LPU.

## Роль в проекте

Groq ключ **обязателен всегда** — используется для двух задач:
1. **OCR картинок** — распознавание текста со скриншотов и фото сценариев (Groq Vision)
2. **Анализ разговора** — если выбран провайдер Groq (а не DeepSeek)

## Почему Groq

- Бесплатный tier: ~14 400 запросов/день
- Скорость: 500–800 токенов/сек (кастомные LPU чипы)
- Не нужно своё железо
- Vision API для чтения картинок — бесплатно

## Модели

| Модель | Использование |
|--------|--------------|
| `llama-3.3-70b-versatile` | Основной анализ, лучший русский |
| `llama-3.1-8b-instant` | Быстрые уточняющие вопросы |
| `llama-3.2-11b-vision-preview` | OCR — чтение текста с картинок |

## Получить ключ

1. [console.groq.com](https://console.groq.com) → зарегистрироваться
2. Меню → **API Keys** → **Create API Key**
3. Ключ начинается с `gsk_...`
4. Вставить в настройки интерфейса — сохраняется в localStorage браузера

## Лимиты бесплатного tier

- ~14 400 запросов / день
- 30 запросов / минута
- ~6 000 токенов / минута на 70b модели

Для 20+ проверок в день — хватает многократно.

## OCR картинок (Groq Vision)

Картинки и скриншоты со сценарием распознаются через Groq Vision независимо от выбранного провайдера анализа. Стоимость — ~1 000–2 000 токенов на изображение, входит в бесплатный лимит.

Поддерживаемые форматы: JPG, PNG, WEBP и другие растровые форматы.
