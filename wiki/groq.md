# Groq API

Groq — облачный сервис для быстрого инференса LLM на кастомных чипах LPU.

## Почему Groq

- Бесплатный tier: 6000 запросов/день
- Скорость: 500-800 токенов/сек (быстрее OpenAI)
- Не нужно своё железо

## Модели

| Модель | Использование |
|--------|--------------|
| `llama-3.3-70b-versatile` | Основной анализ, лучший русский |
| `llama-3.1-8b-instant` | Быстрые уточняющие вопросы |
| `llama-3.2-11b-vision-preview` | Чтение текста с картинок (JPG сценарии) |

## Получить ключ

1. [console.groq.com](https://console.groq.com) → Create API Key
2. Ключ начинается с `gsk_...`
3. Вставить в настройки интерфейса — сохраняется в localStorage браузера

## Лимиты бесплатного tier

- 6000 запросов / день
- 30 запросов / минута  
- ~6000 токенов / минута на 70b модели

Для 20 проверок в день — хватает многократно.
