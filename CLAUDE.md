# whisper-ui

> Контекст для Claude Code. Полная wiki: `wiki-vault/whisper-ui/`.

## Контекст

> Локальный веб-интерфейс для транскрипции через OpenAI Whisper.

## Стек

- Backend: Python + Whisper (OpenAI)
- Frontend: HTML/JS
- Deployment: Docker Compose (локально)

## Команды

```bash
docker compose up
# или
docker-compose -f whisper.yml up
```
